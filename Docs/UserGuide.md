## Setting Up Your Trip

### Add Players

1. Go to **Players** tab
2. Tap **"+ Add Player"**
3. Enter name and handicap index
4. Repeat for all players (you can add more later)

### Assign Teams

1. Go to **Teams** tab
2. Drag players to Team A or Team B
3. Or use **"Auto-Balance"** to split by handicaps

### Add Courses

**Use a Saved Course:**

1. Go to **Courses** tab
2. Tap **"Add from Library"**
3. Select a previously saved course

**Create New Course:**

1. Tap **"+ Add Course"**
2. Enter course name
3. Add tee sets with rating/slope
4. Enter hole pars and handicaps (or use defaults)
5. Tap **"Save to Library"** for future trips

## Creating Sessions & Matches

### Add a Session

1. Go to **Matchups** tab
2. Tap **"+ New Session"**
3. Select format:
   - **Singles** â€” 1v1 match play
   - **Fourball** â€” Best ball (2v2)

### Build Lineups

1. Tap a session card

4. Review the **Fairness Score** (aim for 70+)
5. Tap **"Publish Lineup"** when ready

Once scoring begins:

## Live Scoring

### Score a Match

1. Go to **Score** tab
2. Tap an active match
3. For each hole:
   - Tap **Team A** if they win the hole
   - Tap **Team B** if they win
   - Tap **Halved** for a tie
4. The match status updates automatically:
   - "Team A 2 UP with 5 to play"
   - "DORMIE" when lead equals holes remaining
   - "Team A wins 3&2" on closeout

## Troubleshooting

### Data Not Saving?

- Make sure you're using the app from home screen (not browser)
- Check that your device has storage space
- Try refreshing the page

### Scoring Buttons Not Working?

- Check if the session is locked
- Verify you're on the correct match
- Try disabling one-handed mode temporarily

### Can't Reopen a Match?

- Only captains can reopen matches
- Hold the unlock button for 1.5 seconds
- Check the audit log for recent changes

### Lost Your Data?

- Check if you have a backup export
- Data is stored per-browser â€” try the same browser
- Clearing browser data will delete trip data
